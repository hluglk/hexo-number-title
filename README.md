## hexo-number-title
use timestamp generated value for the permalink `http://hluglk.top/archives/1500846116/`

## usage

### install
```bash
# npm
npm install hexo-number-title --save
# yarn 
yarn add hexo-number-title
```
### configuration
`_config.yml`
``` yml
number_title:
  enable: true
```
`scaffolds/post.md`
```
date: {{ date }}
```
### command
```bash
hexo n 'test'
```
### EXAMPLE
[VCGPGSZO](http://hluglk.top)
