## hexo-number-title
use timestamp generated value for the permalink `http://hluglk.top/archives/1500846116/`

## usage

### install
```bash
npm install hexo-number-title --save
```
### add in `_config.yml`
``` yml
number_title:
  enable: true
  # before_post_render: true # if you want to generate all, value is set to true
```
### command
```bash
hexo clean
hexo g
hexo n
```
